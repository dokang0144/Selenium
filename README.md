# 셀레니움을 활용한 웹 매크로

그래픽카드를 특가로 사기 위해 7일간 셀레니움 공부를 하게 되었습니다...

## 준비하기

### 1. Selenium 설치
+ pip 명령어를 사용해서 설치
```
pip install selenium
```
### 2. WebDriver 설치
+ 크롬 버전 확인
  + ``` 브라우저 → 메뉴 → 도움말 → Chrome 정보``` 로 크롬 버전을 확인한다.
